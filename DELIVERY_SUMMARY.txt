QUICK START COMMANDS
═════════════════════════════════════════════════════════════════════════

1. Install dependencies:
   npm install

2. Run development server:
   npm run dev
   → Open http://localhost:3000

3. Edit content in components/sections/

4. Build for production:
   npm run build
   npm start

5. Deploy to Vercel:
   Push to GitHub
   → Go to vercel.com/new
   → Select your repo
   → Deploy
