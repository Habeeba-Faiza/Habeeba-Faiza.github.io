## üîß Customization

### Update Personal Information

Edit the following in `index.html`:

1. **Contact Information**
   - Email: Find `habeeba.faiza@example.com` and replace
   - Phone: Find `+1 234 567 890` and replace
   - LinkedIn: Update LinkedIn profile URL

2. **Profile Content**
   - Headline, descriptions, and achievements
   - Experience timeline details
   - Skills and services

3. **Contact Methods**
   - Email link
   - LinkedIn profile URL
   - WhatsApp number

### Modify Colors

Edit `:root` variables in `style.css`:

```css
:root {
    --primary-color: #1e3a5f;     /* Change primary color */
    --secondary-color: #2d5a8c;   /* Change secondary color */
    --accent-color: #3b7ec4;      /* Change accent color */
    /* ... other variables ... */
}
```

### Add a Profile Image

1. Replace the SVG placeholder in the hero section with an image
2. Update the `.hero-placeholder` styling as needed
3. Optimize image size for web

## üöÄ Deployment

### GitHub Pages

1. Push all files to your GitHub repository
2. Go to repository Settings ‚Üí Pages
3. Set source to "Deploy from a branch"
4. Select `main` branch and `/root` folder
5. Click Save
6. Your site will be live at `https://yourusername.github.io`

### Local Development

Simply open `index.html` in a web browser. No build process or server required.
